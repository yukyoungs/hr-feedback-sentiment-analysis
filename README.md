# HR 피드백 감성 분석 (HR Feedback Sentiment Analysis)

HR 피드백 텍스트를 Hugging Face 모델로 감성 분석하고 SQLite DB에 저장하는 미니 프로젝트

## 사용 기술
- Python
- Hugging Face Transformers (cardiffnlp/twitter-xlm-roberta-base-sentiment)
- pandas
- SQLite

## 주요 기능
- HR 피드백 텍스트 데이터 전처리
- 다국어 감성 분석 모델로 긍정/부정 분류
- SQLite 데이터베이스에 분석 결과 저장
- 감성별 통계 집계
